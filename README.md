## Setup

To run this project, clone the repository:

```bash
   git clone https://github.com/j45t7/select-pokemon.git
   cd select-pokemon
```

```
$ cd ../select-pokemon
$ npm install
$ npm run dev
```

To run test:

```
$ cd ../select-pokemon
npm test
```
